# 🎨 AI Canvas

An interactive AI-powered virtual drawing application that allows users to draw on a digital canvas using **hand gestures** detected through a webcam. The project leverages **Computer Vision** and **Machine Learning** to provide a touch-free drawing experience.

## 🚀 Features

- ✋ Hand gesture recognition using MediaPipe
- 🖌️ Virtual drawing with finger tracking
- 🎨 Multiple color selection
- 📏 Adjustable brush size
- 🧽 Eraser tool
- 🖼️ Real-time webcam canvas
- 💾 Save drawings as image files
- ⚡ Smooth and responsive drawing experience

## 🛠️ Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/AI-Canvas.git
```

### 2. Navigate to the project folder

```bash
cd AI-Canvas
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

## ▶️ Run the Project

```bash
python main.py
```

## ✨ How It Works

1. Open the application.
2. Allow webcam access.
3. Show your hand to the camera.
4. Use your index finger to draw.
5. Select tools and colors from the toolbar.
6. Save your artwork when finished.
